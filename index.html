概要

このボットは Discord サーバーと Minecraft 統合版（Bedrock Dedicated Server）を連携します。チャット中継、サーバー起動通知、権限連携などの機能を提供します。

Bot を招待する GitHub リポジトリ 導入ガイド
導入ガイド
  1. Discord Developer Portal でアプリを作成し、Bot を有効化します。
  2. 上の招待リンクの CLIENT_ID と PERMISSIONS を置き換えます。
  3. BDS 側にアドオンを導入し、Bot のトークンを設定します（トークンは安全に保管してください）。
  4. 必要に応じて Webhook や設定ページで連携を調整します。

権限は最小限に留め、トークンは公開リポジトリに置かないでください。
